Ejercicio #2: PROMEDIO DE 3 NÚMEROS

Escriba un algoritmo que pregunte por 3 números y muestre como resultado el promedio de estos. Use Pseudocódigo y diagrama de flujo.

Pseudocodigo

Variable Descripción Tipo
A Primer número Float o Int
B Segundo número Float o Int
C Tercer número Float o Int
R Resultado de promedio de A, B y C Float o Int
  1. Inicio
  2. Leer A, B ,C
  3. Promediar A, B y C = Define R
  4. Mostrar R
  5. Fin
