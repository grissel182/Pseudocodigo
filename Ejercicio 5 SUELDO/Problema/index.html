Ejercicio #5: SUELDO

Se requiere determinar el sueldo semanal de un trabajador con base en las horas que trabaja y el pago por hora que recibe. Realice el diagrama de flujo y el pseudocódigo que representen el algoritmo de solución correspondiente

Pseudocodigo

Variable Descripción Tipo
H Horas que ha trabajado Float o Int
P Salario recibido por hora de trabajo Float o Int
S Sueldo semanal basado en las horas y el salario por hora. Float o Int
  1. Inicio
  2. Definir H, P
  3. H * P = S
  4. Mostrar S
  5. Fin
