Ejercicio #7: PINTURAS "LA BROCHA GORDA"

Pinturas “La brocha gorda” requiere determinar cuánto cobrar por trabajos de pintura. Considere que se cobra por m2 y realice un diagramade flujo y pseudocódigo que representen el algoritmo que le permita ir generando presupuestos para cada cliente.

Pseudocodigo

Variable Descripción Tipo
M Número de metros cuadrados Float o Int
P Precio por metro cuadrado Float o Int
R Resultado de precio por M2 Float o Int
  1. Inicio
  2. Definir M * P
  3. M * P = R
  4. Mostrar R
  5. Fin
