Ejercicio #1: SUMA DE DOS NÚMEROS

Escriba un algoritmo que pregunte por dos números y muestre como resultado la suma de estos.

Pseudocodigo

Variable Descripción Tipo
A Primer número a sumar Int
B Segundo número a sumar Int
R Resultado de suma A+B Int
  1. Inicio
  2. Leer A, B
  3. Sumar A + B = Define R
  4. Mostrar R
  5. Fin
